Git Commands
Clone the repo: git clone https://github.com/WiktorBor/AI-group-project.git
Creating a branch: git branch <branchname>
Going to that branch: git checkout <branchname>
Stage a file: git add filename
Stage all files: git add .
Committing change: git commit -m “<changes>”
Push: git push when on your specific branch
